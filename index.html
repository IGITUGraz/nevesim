Welcome to the home of NEVESIM: Neural event-based simulator

  • About

  • NEVESIM (Neural EVEnt-based SIMulator) is a software package for event-driven simulation of spiking neural networks.
    The simulator is written in C++ with a primary interface to Python, intended to be an easily-extendable general purpose tool for event-based neural simulations simple to use.

  • NEVESIM is a free software package available under the GNU General Public License, developed at Institute for Theoretical Computer Science, TU Graz.

  • To learn more about NEVESIM we refer you to documentation and demo pages, but we also encourage you to download NEVESIM and try it yourself! If you have any question regarding NEVESIM consult the FAQ list or feel free to contact us!

  • Features

  • Event-driven simulation of spiking neural networks
  • Easy to use Python interface
  • Supports different levels of modeling
  • Object oriented design
  • Fast C++ core
  • Running on Linux and Windows systems (see dependencies)

  • Why using NEVESIM?

  • NEVESIM was primarily designed and developed to simulate exactly and efficiently networks of stochastic spiking neurons that perform Neural Sampling. To find out more about this class of networks we refer to the demo site.

  • NEVESIM also supports simulation of heterogenerous networks with different types of neurons and synapses, and can be easily extended by the user with new neuron and synapse types.

  • In addition, NEVESIM besides being an event-driven simulator also supports time-step simulation, in this way offering an unique environment for both stiles of simulation.

  • NEVESIM is open source software!
